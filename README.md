# Genny Control Centre

A Command Line Tool used to interact with a local genny system to assist in development.

## Build

```shell
./build.sh
```

## Installing

```shell
./install.sh
```

## Examples

### Cache

```shell
gctl read cache PER_USER
```

```shell
gctl watch db PER_USER
```

```shell
gctl get token
```

### Genny

```shell
gctl status
```

```shell
gctl clone
```

```shell
gctl pull
```

```shell
gctl build
```
